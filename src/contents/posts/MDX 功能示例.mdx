# MDX 功能示例

这篇文章用于验证 MDX 在主题中的表现，包括 Markdown 与 JSX 的混排能力。

## 1. Markdown 语法仍然可用

- 普通列表
- **加粗文本** 与 _斜体文本_
- 行内代码 `pnpm dev`
